Учебные направления

WEB-1

Веб-дизайн, первая чать - изучение основ создания веб-страниц, взаимодествия с ними. Базовые конструкции языка HTML, подключение и настрока стилей CSS.

WEB-2

Веб-дизайн, вторая чать - изучение основ создания красивых сайтов, настрока шрифтов, картинок, расположения компонентов. Ещё большее погружение в стили, изучение фреймворков, технологий вёрстки.

JavaScript

Изучение самого популярного, практически единственного языка программирования для клиентской части WEB, на котором можно также программировать и серверную часть, и десктопные приложения.

PHP

Изучение самого популярного серверного языка программирования для создания интернет-магазинов и социальных сетей. VK, Facebook написаны на PHP.
